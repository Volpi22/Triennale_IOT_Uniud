# Bachelor's Degree in Internet of Things, Big Data, and Machine Learning  
**University of Udine**  

Welcome to my GitHub repository documenting my academic journey during my Bachelor's degree in **Internet of Things (IoT), Big Data, and Machine Learning** at the University of Udine. This repository contains organized notes, projects, and resources from my three years of study. The goal of this repository is to provide a structured and comprehensive overview of the topics covered during my degree, as well as to serve as a reference for others interested in these fields.

---

## Repository Structure  
The repository is organized into three main folders, corresponding to each academic year. Below is a detailed breakdown of the contents:

### **1_Anno (First Year)**  
- **Fisica**: Notes and resources related to Physics, covering fundamental principles and their applications in technology.  
- **Programmazione**: Introduction to programming concepts, focusing on foundational languages and problem-solving techniques.  
- **Scienze dei dati e Web**: Basics of data science and web technologies, including data analysis, visualization, and web development fundamentals.  

### **2_Anno (Second Year)**  
- **Algoritmi**: Study of algorithms, including design, analysis, and optimization techniques.  
- **Machine_Learning**: Introduction to machine learning concepts, algorithms, and practical applications.  
- **Programmazione ad oggetti**: Object-oriented programming principles and practices, with examples in relevant programming languages.  
- **Sistemi operativi**: Operating systems concepts, including process management, memory management, and file systems.  
- **Statistica**: Statistical methods and their applications in data analysis and machine learning.  
- **Tecnologie_Web**: Advanced web technologies, including frameworks, APIs, and modern web development practices.  

### **3_Anno (Third Year)**  
- **Basi**: Foundational topics in computer science and engineering, reinforcing core concepts.  
- **DTEH**: Data Engineering and High-Performance Computing, focusing on large-scale data processing and optimization.  
- **IUM**: Intelligent User Interfaces and Human-Computer Interaction.  
- **Ingegneria del software**: Software engineering principles, including design patterns, development methodologies, and project management.  
- **IoT**: Internet of Things, covering IoT architectures, protocols, and applications.  
- **Reti**: Networking fundamentals, including protocols, security, and distributed systems.  
- **SocialComputing**: Social computing and its applications in data analysis, machine learning, and user behavior modeling.  
- **Thesis**: My Bachelor's thesis.
